Introduction¶

The movie business is an interesting one, and every commercial movie's main goal is to turn a profit. I will be looking at the TMDb movie dataset for my investigation. This data collection includes details on 10,000 movies that were gathered from The Movie Database (TMDb), including genre and revenue information that will serve as the foundation for this investigation. some questions have been asked and will be answered using the following tools:

  • python
  • matplotlib
  • numpy

Question to be answered¶

  • Which genres are the most popular year after year?

  • Is there a link between popularity and high-profit films?

  • Which months have the highest film profits?

  • Which actor has the most movie popularity and profit?

  • How does the length of a film effect its profit?

My Finding

We can see the shape, description, and information for each column in our dataset here, and there are some interesting things to highlight.

The dataset has 10866 rows and 21 columns; there appear to be some missing values in nearly half of the columns; the homepage column has 7930 missing values, followed by the tagline column with 2824.

According to the dataframe's description, almost half of the budget and revenue columns contain the value $0 (assuming the values are in dollars), which I believe is missing data. A couple movies have 0 minutes and one has 900 minutes, which appears to be an error. A movie's average runtime is 102 minutes.

About 50% of the movies are with the years 2006-2015 and the relese_date data type should be in datetime format.

columns to drop are homepage, id, imdb_id, tagline, keywords, overview, budget_adj, revenue_adj, vote_count, vote_average.

note: The vote count and vote average columns should be removed because inconsistency in the vote count affects the integrity of the vote average, which is dependent on the vote count. i.e A vote total of only ten people rating a film at 8.0 is not a strong indicator of quality.

#This function will create all vertical barplots that apply the groupby function.

def bar_plot_groupby(group_by, column, func, title=None, xlabel=None, ylabel=None, ticks=None, labels=None, figsize=None):
    """
    groupby argument takes the column the dataframe should be grouped by
    the column argument takes the name of the column to be used
    the title argument returns the title of the plot
    and func is the function you want to perform on the column, which can either be 'count', np.mean, np.sum
    other arguments that can be used are xlabel, ylabel, ticks and labels for the x-axis
    """
    if func == 'count': 
        df_plot = df_movies_edit.groupby(group_by)[column].count()
    else:
        df_plot = df_movies_edit.groupby(group_by)[column].agg(func)
    df_plot.plot(kind='bar',fontsize=13, xlabel=xlabel, ylabel=ylabel, figsize=figsize)
    plt.xticks(rotation=45)
    if ticks != None and labels!= None:
        plt.xticks(ticks=ticks,labels=labels, rotation='horizontal')
    plt.title(title,fontsize=14)

#store the range of years in the year variable
year = [range(df_movies_edit.release_year.min(),df_movies_edit.release_year.max()+1)]
#loop through each year and split the genre into individual components
for i in range(df_movies_edit.release_year.min(),df_movies_edit.release_year.max()+1):
    df_new = df_movies_edit.query('release_year == {}'.format(i))
    genre = df_new.genres.str.split('|')
    dic_genre = {}
    for j in genre:
        for x in j:
            keys = dic_genre.keys()
            if x in keys:
                dic_genre[x] += 1 
            else:
                dic_genre[x] = 1
    
    df_year_genre.append(dic_genre)
    #return the genre with the maximum number of movies
    maximum = max(dic_genre, key= dic_genre.get)
    year_genre.append([i,maximum]) 
    dic_genre["year"] = i

Conclusions¶

The following are the findings of the research:

  • The drama genre is the most produced year after year, according to the first question.

  • Based on the second question, popularity is a deciding factor in movie profit, however this is not always the case due to lurking variables.

  • In response to the third question, the month of June has been the most profitable period to release a movie.

  • From the fourth question, we can see the top 25 actors with the most appearances, and we can also see that Harrison Ford is in the most popular and profitable movies.

  • Finally, most films are released in the 90-120 minute range, although those in the 150-180 minute range appear to make more money on average.

Limitations:

  • Due to null values a lot of rows were dropped
  • zero values in the budget and revenue columns
